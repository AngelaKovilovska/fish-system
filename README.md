# CLARIO — Систем за управување со рибник

Веб-апликација за евиденција и управување со рекиркулационен аквакултурен систем (RAS) за одгледување на африкански сом (*Clarias gariepinus*), развиена за **Фамаком Аквакултура**.

**Продукција:** [clario.up.railway.app](https://clario.up.railway.app)

---

## Опис

CLARIO овозможува целосна дигитализација на секојдневните операции во рибникот — од дневна контрола на системот, преку хранење и мерење на водата, до AI-базирани препораки за хранење и предвидување на квалитетот на водата.

Апликацијата е дизајнирана за мобилен пристап (PWA) бидејќи операторите работат покрај базените и немаат пристап до компјутер.

## Функционалности

### Дневни операции
- **Дневна чеклиста** — проверка на биофилтер, механички филтер, пумпи, аерација, состојба на рибите, параметри на водата (температура, pH, нитрати, нитрити, амониум, алкалитет, тврдост, хлор)
- **Хранење (3 оброка/ден)** — внес по базен со поддршка за повеќе типови храна, автоматско пресметување на залихи
- **Мерење на риби** — просечна тежина и број по базен, историја на мерења

### Аларми и известувања
- Автоматско генерирање на аларми кога параметрите излегуваат од нормата
- Преглед на активни аларми на почетната страна
- Масовно означување како прочитано

### Залихи на храна
- Евиденција на набавки (датум, добавувач, количина, документ)
- Автоматско пресметување на моментален залих врз основа на набавки минус потрошувачка
- Проекција за кога ќе завршат залихите

### AI модул — Препораки за хранење
- Пресметка на оптимална дневна количина храна по базен
- Базирано на: број на риби, просечна тежина, температура на вода, тип на храна
- Табела на хранење по Coppens 2025-2026 спецификации
- Споредба на реално дадена храна vs. препорачана

### AI модул — Предвидување на квалитет на вода
Систем од 5 компоненти базиран на научна литература за *Clarias gariepinus*:

1. **NH₃ Калкулатор** — пресметка на токсичен амонијак врз основа на pH и температура (Emerson et al., 1975)
2. **Z-Score детекција** — откривање на ненормални вредности преку статистичка анализа
3. **Тренд предвидувач** — линеарна регресија за предвидување кога параметар ќе излезе од норма
4. **Каузален ланец** — поврзување на причина и последица (пр. прехранување → амонијак → нитрити)
5. **Препораки** — конкретни чекори за операторот со дозировки и извори од научна литература (Boyd 1998, Schram et al. 2014, Timmons & Ebeling 2010, SRAC)

### Извештаи
- Потрошувачка на храна по период и базен
- Просечни тежини по базен
- Извештај за аларми
- Сортирања
- Набавки на храна

### Администрација
- Управување со корисници (admin/работник)
- Поставување на норми за параметрите на водата
- Управување со мерења и залихи

## Инсталација (локален развој)

### Предуслови
- Node.js >= 18
- PostgreSQL >= 14

### Чекори

```bash
# 1. Клонирај го репото
git clone https://github.com/AngelaKovilovska/fish-system.git
cd fish-system

# 2. Инсталирај зависности
npm run install:all

# 3. Креирај .env фајл во server/
cat > server/.env << EOF
DATABASE_URL=postgresql://user:pass@localhost:5432/clario
JWT_SECRET=your-secret-key
NODE_ENV=development
PORT=3001
EOF

# 4. Креирај база и пушти миграции (автоматски при старт)
createdb clario

# 5. Стартувај
npm run dev
```

Апликацијата ќе биде достапна на `http://localhost:5173` (client) и `http://localhost:3001` (API).

При прв старт автоматски се креира admin корисник:
- Email: `admin@clario.mk`
- Лозинка: `admin123`

## API

Серверот нуди REST API со 11 модули:

| Модул | Патека | Опис |
|-------|--------|------|
| Auth | `/api/auth` | Најава, одјава, сесија |
| Records | `/api/records` | Дневни чеклисти |
| Alerts | `/api/alerts` | Аларми (преглед, потврдување) |
| Meals | `/api/meals` | Оброци по базен |
| Food Inventory | `/api/food-inventory` | Залихи, набавки, проекции |
| Pool Measurements | `/api/pool-measurements` | Мерења на риби |
| Pool Fish Inventory | `/api/pool-fish-inventory` | Моментален број/тежина по базен |
| Norms | `/api/norms` | Норми за параметри на вода |
| Users | `/api/users` | Управување со корисници |
| Reports | `/api/reports` | Генерирање на извештаи |
| AI | `/api/ai` | Препораки, калкулатор, предвидувања |

Сите endpoints бараат JWT автентикација (освен `/api/auth/login`).

## База на податоци

13 миграции кои се извршуваат автоматски при старт на серверот:

- `001_initial.sql` — корисници, дневни записи, водена контрола, аларми, норми
- `006_pool_meals.sql` — оброци по базен
- `008_pool_fish_inventory.sql` — инвентар на риби по базен
- `010_water_params_update.sql` — алкалитет, хлор, амониум
- `011_pool_meals_multi_food.sql` — повеќе типови храна по оброк
- `013_expand_pools_to_8.sql` — проширување на 8 базени
